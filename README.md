# MyPlayer
 功能介绍如下：             
  1、获取本地歌曲列表，实现歌曲播放功能。             
  2、利用jsoup解析网页数据，从网络获取歌曲列表，同时实现歌曲和歌词下载到手机本地的功能。         
  3、通知栏提醒，实现仿QQ音乐播放器的通知栏功能.              
  涉及的技术有：        
  1、jsoup解析网络网页，从而获取需要的数据        
  2、android中访问网络，获取文件到本地的网络请求技术，以及下载文件到本地实现断点下载        
  3、线程池         
  4、图片缓存        
  5、service一直在后台运行         
  6、Activity与Fragment间的切换以及通信         
  7、notification通知栏设计         
  8、自定义广播         
  9、android系统文件管
